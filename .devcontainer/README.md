# Dev Container Configuration

This directory contains the configuration for GitHub Codespaces and VS Code Dev Containers.

## What's Included

- **Docker-in-Docker**: Full Docker support including Docker Compose
- **VS Code Extensions**: Docker extension pre-installed
- **Shell**: Zsh with Oh My Zsh for a better terminal experience

## How It Works

When you open this repository in GitHub Codespaces or a VS Code Dev Container, it will:

1. Create a development environment based on Ubuntu
2. Install Docker and Docker Compose
3. Install helpful VS Code extensions
4. Set up a convenient shell environment

This ensures everyone has the same Docker development experience, regardless of their host operating system.
